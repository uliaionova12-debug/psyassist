## PsyAssist

AI-powered SaaS platform for psychologists.

## About Project

PsyAssist helps psychologists with:

- AI supervision of clinical cases
- content generation from practice
- client hypotheses
- therapeutic planning
- professional growth support

## Features

- Structured AI support for clinical case supervision
- Practice-based content generation and drafting
- Client hypothesis exploration and refinement
- Therapy planning assistance and session preparation
- Tools to support ongoing professional development

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Supabase

## Installation
